# AI Video Landing Page 🎬

A simple, beginner‑friendly landing page built with HTML and CSS to showcase an AI video creation tool.  
The page is structured like a real marketing landing page with a clear story and single call to action.

## 🔥 Features

- Hero section with headline, subheading, and primary CTA button  
- Problem and solution sections to tell a clear story  
- Features/benefits grid (easy to use, fast, no editing skills, works on phone & PC)  
- Social proof section with stats and testimonials  
- CTA section to drive signups or trials  
- FAQ section using native `<details>` for toggles  
- Responsive layout using modern CSS and a clean, dark gradient design  

## 🧱 Tech Stack

- **HTML5** – structure and content  
- **CSS3** – layout, gradients, responsive design  
- No JavaScript required
